# CKAD Study Guide

## Exam Details and Guide

### Exam Objectives
* 13% – Core Concepts

* 18% – Configuration

* 10% – Multi-Container Pods

* 18% – Observability

* 20% – Pod Design

* 13% – Services & Networking

* 8% – State Persistence

### Core Concepts
* Kubernetes Environment - Collection of Objects (Primitives) 

* Creating, Deleting and Modifying Objects using CLI e.g. Creating, Configuring, and Inspecting Pods.

### Configuration
* Configuring Pods with other K8s objects
* ConfigMaps and Secrets
* Privileges and Access Control using Security Context
* ResourceQuota limiting CPU and Memory
* Service Accounts
* K8s objects defining identities for processes running in a Pod

### Multi-container Pods
* Init containers
* Sidecar 
* Adapter
* Ambassador

### Observability
* Health States: Readiness, Liveness, Startup probes
* How to debug containers that failed

### Pod Design
